# 投资思路回测引擎 — 产品设计文档

## 一、核心洞察

传统量化回测只能验证可数值化的交易规则（PE<10买入、均线金叉做多），但大量真实的投资决策依赖定性判断：

- "这家公司的高股息能持续吗？"
- "低PE是真便宜还是价值陷阱？"
- "管理层在做价值创造还是资本运作？"
- "行业格局能支撑这个毛利率吗？"

这些判断无法用因子回测，但可以用 **AI盲测回测** 来验证。

**本项目的核心发现：任何可文字描述的投资思路，都可以通过"结构化框架 + 历史截面盲测 + 统计验证"来检验其有效性。**

## 二、已验证的概念证明

### 2.1 实验设计

| 维度 | 设计 |
|------|------|
| 投资思路 | 龟级价值投资（V5.5.6框架）：低PE/PB+高股息+FCF质量 |
| 回测范围 | 2019-06 ~ 2024-12，12个半年截面 |
| 采样方法 | 每截面从龟级筛选池中选5只：最好2+最差2+中间1 |
| 盲测方式 | 隐藏公司名称，仅提供财务数据+行情数据 |
| AI角色 | 标准化分析师——接收相同框架和数据，产出一致可比的判断 |
| 验证指标 | AI评分/建议 vs 实际6个月前向收益 |

### 2.2 核心结果

| 策略 | 样本 | 平均6m收益 | 胜率 |
|------|------|-----------|------|
| 纯量化龟级筛选 | 60只 | +7.5% | - |
| 量化 + AI过滤（仅执行买入） | 9只 | +24.1% | 67% |
| **AI过滤增益** | - | **+16.6pp** | - |

6年累计净值：纯量化 1.50x，高确信AI 2.01x（仅出手8次）。

### 2.3 AI能力边界（关键发现）

| AI强项 | 证据 |
|--------|------|
| 识别高杠杆价值陷阱 | 地产股18只中16只被标记回避，平均收益-10% |
| 高确信买入信号 | ≥70分买入的6只，平均+41.5%，胜率83% |
| 不出手也是alpha | 2019/2024无买入信号=持现金，规避了下跌 |

| AI弱点 | 证据 |
|--------|------|
| 周期底部反转 | 鲁西化工25分→+73%，苏宁环球37分→+136% |
| 牛市中过于保守 | 2020年回避信号平均收益+30.6%（错过牛市） |
| 银行股区分度低 | 银行普遍60-70分，买入/观望界限模糊 |

### 2.4 按年度表现

| 年份 | 纯量化 | 高确信AI | AI买入明细 |
|------|--------|---------|-----------|
| 2019 | -9.9% | 持现金 | 无信号 |
| 2020 | +28.4% | -13.3% | 交通银行89分（唯一失手） |
| 2021 | +7.7% | +21.4% | 中国神华82分+46%、大秦铁路72分-3.5% |
| 2022 | +0.9% | +28.7% | 中国石化81分+7%、中国石油72分+50% |
| 2023 | +9.5% | +48.4% | 平煤86分+56%、淮北91分+48%、南京银行72分+41% |
| 2024 | +8.8% | 持现金 | 无信号 |

### 2.5 按行业表现

| 行业 | 样本 | 均收益 | AI效果 |
|------|------|--------|--------|
| 地产/建筑 | 18只 | -10.0% | 最强：16只回避，精准规避陷阱 |
| 能源/化工 | 9只 | +10.9% | 次强：4只买入均收益+40% |
| 银行 | 7只 | +8.9% | 良好：方向正确但区分度有限 |
| 钢铁/有色 | 10只 | -13.5% | 一般：偏保守但方向正确 |
| 交运/基建 | 4只 | +32.5% | 盲区：中远海控+61.7%被回避 |
| 其他(周期) | 12只 | +39.9% | 最弱：周期反弹系统性错过 |

## 三、通用化引擎设计

### 3.1 核心抽象

```
投资思路回测 = f(投资框架, 选股范围, 回测参数)

其中：
  投资框架 = 文本描述的投资哲学 + 分析维度 + 评分标准
  选股范围 = 初筛条件 + 市场范围
  回测参数 = 时间跨度 + 截面频率 + 采样策略 + 验证指标
```

### 3.3 关键组件

#### 组件1：框架解析器

将自然语言描述的投资思路转化为结构化的分析Prompt模版。

**输入**：投资思路文本（可以是一段话，也可以是一本书的核心理念）

**输出**：
- 分析维度列表（如：负债质量、现金流、估值、管理层...）
- 每个维度的评分标准
- 综合评分规则
- 买入/回避判定逻辑

**当前实现**：V5.5.6框架手动拆分为10章Prompt（config/framework_chunks/）

**通用化方向**：
- 用AI自动将投资书籍/文章解析为结构化框架
- 提供框架模版库（价值型/成长型/周期型/事件驱动型...）
- 支持用户自定义维度和权重

#### 组件2：截面采样器

决定"在哪些时间点、分析哪些股票"。

**采样策略选项**：

| 策略 | 适用场景 | 说明 |
|------|---------|------|
| 极端采样 | 验证区分能力 | 选最好N只+最差N只+中间，最大化信号 |
| 随机采样 | 验证泛化能力 | 从候选池随机抽取，评估实战效果 |
| 分层采样 | 验证行业适用性 | 按行业/市值/风格分层抽取 |
| 全量采样 | 最终验证 | 候选池全部分析，成本高但结论最强 |

**当前实现**：极端采样（每截面最好2+最差2+中间1）

#### 组件3：数据快照器

生成某时间点的完整数据包，确保时间边界不被违反。

**三层防护**：
1. **数据层硬过滤**：财报按公告日（非报告期）过滤，行情按交易日过滤
2. **Prompt显式注入**：在分析指令中明确截止日期和禁止前瞻
3. **输出校验**：扫描AI输出中的日期引用，标记越界引用

**盲化处理**：
- 隐藏公司名称、股票代码
- 替换为匿名标识符
- 保留所有数值数据

#### 组件4：AI盲测分析引擎

核心执行层。

**关键设计决策**：
- **盲测**：隐藏公司身份，消除AI的品牌偏见和记忆污染
- **标准化**：同一框架+同一数据格式 → 可比较的输出
- **并行化**：60个Agent可同时运行，总耗时约5-10分钟
- **模型选择**：Sonnet级别模型即可（性价比最优），Opus用于框架设计

**当前实现**：
- Claude Code Agent调用Sonnet模型
- 每个分析产出600-900行结构化报告
- 提取综合评分（XX/100）和投资建议（买入/观望/回避）

#### 组件5：验证与归因引擎

将AI判断与实际结果对比，产出统计验证报告。

**验证维度**：

| 维度 | 指标 | 说明 |
|------|------|------|
| 方向正确率 | 高分+正收益 或 低分+负收益 的比例 | 基础有效性 |
| 策略增益 | AI过滤后收益 - 纯量化收益 | 核心alpha |
| 信号质量 | 买入信号胜率、回避信号命中率 | 信号可靠性 |
| 按行业归因 | 各行业的AI准确率 | 能力边界 |
| 按市场环境归因 | 牛市/熊市/震荡市的表现差异 | 适用场景 |
| 假阳/假阴分析 | 错误案例的共性特征 | 改进方向 |

**输出**：
- 结构化验证报告（Markdown）
- 逐年/逐截面收益对比表
- 能力边界地图（强于什么、弱于什么）
- 改进建议（框架哪些维度需要增强）

## 四、可验证的投资思路库

以下思路均可用本引擎验证，每种需要定义其特有的框架和采样策略：

### 4.1 已验证

| 思路 | 框架 | 状态 |
|------|------|------|
| 龟级价值投资 | V5.5.6（10章FCF/负债/分红） | 已验证：+16.6pp增益 |

### 4.2 待验证（按优先级排序）

| 思路 | 核心问题 | 框架化方向 | 初筛条件 |
|------|---------|-----------|---------|
| **高股息陷阱识别** | "高息是真赚钱还是透支未来？" | 分红可持续性、FCF覆盖、负债趋势 | 股息率>5%的全部A股 |
| **困境反转** | "业绩暴雷后能否恢复？" | 现金流拐点、负债变化、管理层行动 | ST摘帽股/业绩预亏后 |
| **周期择时** | "当前处于周期什么位置？" | 库存周期、产能利用率、价格趋势 | 钢铁/化工/航运/养殖等 |
| **成长股合理估值** | "高增长能否支撑高PE？" | PEG、市场空间、竞争壁垒、增速可持续性 | 创业板/科创板PE>30 |
| **央国企价值重估** | "中特估是否有基本面支撑？" | 分红提升空间、治理改善、资产注入 | 央国企控股+PB<1 |
| **小市值困境反转** | "小公司暴跌后是否有反弹机会？" | 壳价值、资产质量、现金流底线 | 市值<50亿+跌幅>50% |

### 4.3 框架设计模版

为新思路编写分析框架时，应包含以下要素：

```markdown
# [投资思路名称] 分析框架

## 核心假设
- 该思路认为什么情况下值得投资？
- 该思路认为什么是最大风险？

## 分析维度（3-8个）
### 维度1: [名称]
- 数据需求：需要哪些数据来评估？
- 评分标准：什么算好、什么算差？
- 权重：在总评分中占多少？

### 维度2: ...

## 综合评分规则
- 总分=100分的各维度权重分配
- 买入/观望/回避的阈值

## 排除条件（一票否决）
- 什么情况下不论评分多高都应回避？

## 已知局限
- 该思路在什么市场环境下可能失效？
```

## 六、风险与限制

| 风险 | 缓解措施 |
|------|---------|
| 幸存者偏差 | 采样时纳入已退市/ST股票 |
| 过拟合框架 | 用样本外数据验证；交叉验证 |
| AI模型偏差 | 多模型对比（Claude/GPT/Gemini）；定期重测 |
| 数据前瞻 | 三层时间边界防护 |
| 采样偏差 | 多种采样策略对比 |
