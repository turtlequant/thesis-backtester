关键发现

  • 排雷价值 > 选股价值：回避信号的 alpha 贡献 (-14.8pp) 远超买入信号 (+6.4pp)，验证了"宁可错过，不可错买"的设计理念
  • Agent 评分有区分度：评分越高，前向收益越好，呈现单调递增关系
  • 深度分析优于简单筛选：Agent Buy vs Screen All 有稳定的超额收益

如何编写算子 (Operator)

算子是分析的最小单元，定义在 operators/v2/ 目录下的 Markdown 文件中。每个算子包含 YAML 前置元数据和 Markdown 分析指引。

文件格式

---
id: my_operator
name: 我的分析算子
category: fundamental
tags: [analysis, my_tag]
data_needed: [balancesheet, income, cashflow]
outputs:
  - field: my_result
    type: str
    desc: "分析结论"
  - field: my_score
    type: float
    desc: "评估分数 (0-100)"
---

## 算子标题

### 目的
说明这个算子要回答什么问题...

### 分析步骤
具体的分析指引，Agent 会按照这些步骤执行分析...

关键字段说明

  • id: 唯一标识符，在 strategy.yaml 中引用
  • category: 分类 (screening / fundamental / valuation / decision / forward_risk / special)
  • data_needed: 需要的数据表 (balancesheet, income, cashflow, fina_indicator, price_history, dividend, news 等)
  • outputs: 输出字段定义，Engine 会自动生成 JSON Schema
  • gate (可选): 行业限制，如 gate: {only_industry: [银行]}

如何创建策略 (Strategy)

策略是算子的编排方案，定义在 strategies/<name>/strategy.yaml 中。

基本结构

meta:
  name: 我的策略
  version: "1.0"

framework:
  operators_dir: operators/v2
  analyst_role: 投资分析师

  chapters:
  - id: ch01_check
    chapter: 1
    title: 数据核查
    operators: [data_source_grading, quick_screen_5min]
    dependencies: []

  - id: ch02_analysis
    chapter: 2
    title: 基本面分析
    operators: [debt_structure, cash_trend_5y]
    dependencies: [ch01_check]

  synthesis:
    thinking_steps:
      - step: 评分与决策
        instruction: 综合分析给出评分...
    decision_thresholds:
      buy: 75
      avoid: 29

设计要点

  • 章节间通过 dependencies 形成 DAG，后续章节可看到前序章节结论
  • synthesis 定义综合研判的思考路径和评分锚点
  • 可从现有策略 (v6_value, v6_enhanced) 复制并修改
  • 支持行业专用策略 (如 bank_analysis)，替换通用算子为行业专用算子

如何提交贡献

  1. Fork 仓库到你的 GitHub 账号
  2. 创建特性分支: git checkout -b feature/my-operator
  3. 添加或修改算子/策略文件
  4. 本地测试: 
    # 测试算子能否被正确加载
python -m src.engine.launcher strategies/v6_enhanced/strategy.yaml agent-analyze 601288.SH 2024-06-30

# 重新生成站点数据
python docs/site/build.py
  5. 提交 PR，说明变更内容和测试结果

贡献方向

  • 新行业专用算子 (地产、医药、新能源 ...)
  • 新分析维度 (ESG、技术面、量价关系 ...)
  • 策略优化 (调整阈值、改进 synthesis 思考步骤)
  • 国际化 (港股、美股适配)
