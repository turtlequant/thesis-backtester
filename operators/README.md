# 算子库 (Operators)

算子是分析方法论的最小单元。每个算子是一个 `.md` 文件，定义了一个独立的分析视角——LLM 按照算子指令执行结构化分析，产出可验证的结论。

## 版本

| 版本 | 算子数 | 状态 | 说明 |
|------|--------|------|------|
| [v1/](v1/) | 21 | 冻结 | 与回测结果绑定（+7.1pp alpha），不可修改 |
| [v2/](v2/) | 26 | 活跃 | v1 全部 + 5 个前瞻风险算子 + 输出增强 + 行业门控 |

v1 永不修改，确保回测结果可复现。新增和优化都在 v2 进行。

## 策略如何引用

```yaml
# strategy.yaml
framework:
  operators_dir: operators/v1    # 锁定 v1
  # 或不指定，默认使用 v2
```
