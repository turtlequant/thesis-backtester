## 资产质量与不良率分析

资产质量是银行最核心的基本面指标。不良贷款率和拨备覆盖率决定了银行利润的真实性和未来的损失吸收能力。

### 一、不良贷款率（NPL Ratio）趋势分析

**步骤1：获取近5年不良贷款率数据**
- 从 fina_indicator 获取历年不良贷款率
- 绘制趋势图：持续下降（正面）、波动平稳（中性）、持续上升（负面）

**步骤2：不良贷款率绝对水平判断**

| 不良率水平 | 判断 | 说明 |
|-----------|------|------|
| < 1.0% | 优秀 | 资产质量上乘，但需检查是否通过转让/核销美化 |
| 1.0% - 1.5% | 良好 | A股银行正常水平 |
| 1.5% - 2.0% | 需关注 | 高于行业平均，分析原因 |
| > 2.0% | 警告 | 资产质量明显承压 |

**步骤3：不良变动拆解**
```
期末不良 = 期初不良 + 新生成不良 - 核销及转出 - 回收
不良净生成率 = (新生成不良 - 回收) / 平均贷款余额
```
- 不良净生成率上升 → 资产质量真正恶化
- 不良率下降但核销力度加大 → 可能是"以核销换指标"，需警惕

### 二、拨备覆盖率分析

**步骤1：拨备覆盖率计算与判断**
```
拨备覆盖率 = 贷款减值准备余额 / 不良贷款余额 × 100%
```

| 拨备覆盖率 | 判断 | 说明 |
|-----------|------|------|
| > 300% | 过度拨备 | 可能存在"藏利润"空间，拨备释放是潜在利好 |
| 200% - 300% | 充足 | 损失吸收能力强 |
| 150% - 200% | 达标 | 监管底线附近，缓冲有限 |
| < 150% | 不足 | 低于监管要求（一般150%），利润可能被拨备计提侵蚀 |

**步骤2：拨备覆盖率与不良率的联动**
- 不良率上升 + 拨备覆盖率下降 → 最危险的组合，说明拨备计提跟不上不良生成
- 不良率下降 + 拨备覆盖率上升 → 最正面的信号，资产质量真正改善
- 不良率上升 + 拨备覆盖率也上升 → 管理层主动加大拨备力度，态度积极
- 不良率下降 + 拨备覆盖率也下降 → 可能通过释放拨备虚增利润

**步骤3：拨贷比交叉验证**
```
拨贷比 = 贷款减值准备余额 / 贷款总额 × 100%
监管要求：≥ 2.5%（部分银行可适用1.5%过渡期标准）
```
- 拨贷比更不易被操纵（分母是全部贷款），是拨备覆盖率的重要补充

### 三、信用成本分析

**计算公式**：
```
信用成本 = 当期资产减值损失（含信用减值损失） / 平均贷款余额 × 100%
```

**判断标准**：

| 信用成本 | 判断 | 说明 |
|---------|------|------|
| < 0.5% | 低 | 资产质量优秀或拨备释放 |
| 0.5% - 1.0% | 正常 | A股银行常见水平 |
| 1.0% - 1.5% | 偏高 | 资产质量承压，侵蚀利润 |
| > 1.5% | 高 | 大幅计提，利润可能大降 |

**趋势分析**：
- 信用成本持续下降 → 资产质量改善的领先信号（比不良率更早反映）
- 信用成本突然跳升 → 可能集中暴露风险，检查是否为一次性出清

### 四、关注类贷款与迁徙率

**关注类贷款**是不良的"蓄水池"：
```
关注类贷款占比 = 关注类贷款 / 贷款总额
关注类→不良迁徙率 = 期内从关注类下迁至不良的金额 / 期初关注类余额
```

**警示信号**：
- 关注类占比 > 3% → 未来不良上升压力大
- 关注类→不良迁徙率 > 30% → 关注类中"准不良"占比高
- 关注类占比下降但不良率未同步下降 → 可能直接从正常类跳入不良（管理漏洞）

### 五、贷款集中度风险

**行业集中度**：
- 房地产贷款占比：>15%需重点关注，当前环境下是核心风险点
- 制造业贷款占比：周期性强，关注行业景气度
- 基建/政府平台贷款：关注地方债务风险区域

**客户集中度**：
- 最大单一客户贷款占资本净额比例（监管要求 ≤10%）
- 最大十家客户贷款占资本净额比例（监管要求 ≤50%）
- 集中度越高，单一违约对资产质量冲击越大

**区域集中度**：
- 城商行、农商行的区域集中度天然较高
- 关注所在区域的经济增速、房价走势、产业结构
- 区域经济下行 → 区域性银行不良率可能快速攀升

### 六、逾期贷款与不良偏离度

**不良偏离度**：
```
不良偏离度 = 逾期90天以上贷款 / 不良贷款
```

| 偏离度 | 判断 | 说明 |
|--------|------|------|
| < 80% | 严格 | 不良认定严格，资产质量真实可信 |
| 80% - 100% | 合理 | 基本匹配 |
| > 100% | 宽松 | 存在应列未列不良，真实不良率更高 |

**关键提醒**：2019年后监管要求逾期60天以上纳入不良，偏离度>100%的银行需重点警惕。
