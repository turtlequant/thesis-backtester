## 银行压力测试

压力测试评估银行在极端情景下的资本充足性和分红可持续性。核心问题：如果不良率大幅上升，银行能否在不触发监管底线的情况下继续经营和分红？

### 一、资本充足率现状

**步骤1：获取资本指标**
- 从 fina_indicator 获取最新资本充足率相关数据
- 核心一级资本充足率（CET1 Ratio）
- 一级资本充足率（Tier 1 Ratio）
- 资本充足率（CAR）

**监管最低要求**（2024年标准）：

| 指标 | 最低要求 | 储备要求 | 系统重要性附加 | 合计（大行） | 合计（中小行） |
|------|---------|---------|---------------|-------------|---------------|
| CET1 | 5.0% | 2.5% | 0.25%-1.5% | 8.0%-9.5% | 7.5% |
| Tier1 | 6.0% | 2.5% | 0.25%-1.5% | 9.0%-10.5% | 8.5% |
| CAR | 8.0% | 2.5% | 0.25%-1.5% | 11.0%-12.5% | 10.5% |

**步骤2：资本缓冲计算**
```
CET1缓冲 = 实际CET1比率 - 监管最低要求（含附加）
```
- 缓冲 > 3个百分点 → 资本非常充裕
- 缓冲 1.5-3个百分点 → 资本较充裕
- 缓冲 0.5-1.5个百分点 → 资本偏紧
- 缓冲 < 0.5个百分点 → 资本紧张，可能被迫限制分红或融资

**步骤3：系统重要性银行识别**
- 全球系统重要性银行（G-SIB）：中工建农交（附加0.5%-1.5%）
- 国内系统重要性银行（D-SIB）：约20家，附加0.25%-1.0%
- 确认目标银行的系统重要性等级及对应的资本附加要求

### 二、不良率上升压力测试

**核心假设**：如果不良率上升100bp或200bp，银行需要计提多少额外拨备，对资本的冲击有多大？

**步骤1：基准情景参数**
```
从最新报表获取：
- 贷款总额 L
- 当前不良贷款率 NPL_0
- 当前拨备覆盖率 PC_0
- 当前不良贷款余额 = L × NPL_0
- 当前拨备余额 = 不良贷款余额 × PC_0
```

**步骤2：轻度压力情景（NPL +100bp）**
```
新不良余额 = L × (NPL_0 + 1%)
新增不良 = L × 1%
维持拨备覆盖率150%所需拨备 = 新不良余额 × 150%
额外拨备计提 = max(所需拨备 - 当前拨备余额, 0)
税后利润冲击 = 额外拨备计提 × (1 - 25%)
CET1影响 = 税后利润冲击 / 风险加权资产
```

**步骤3：重度压力情景（NPL +200bp）**
- 同上计算方法，使用NPL_0 + 2%
- 重度情景模拟经济深度衰退或房地产硬着陆

**步骤4：极端压力情景（NPL翻倍）**
- NPL翻倍情景代表系统性金融危机
- 计算此情景下CET1是否仍高于监管底线

**压力测试结果判断**：

| 情景 | CET1仍达标 | CET1跌破 | 说明 |
|------|-----------|---------|------|
| 轻度（+100bp） | 通过 | 极其危险 | 正常银行应轻松通过 |
| 重度（+200bp） | 较安全 | 需关注 | 优质银行应通过 |
| 极端（翻倍） | 非常安全 | 可接受 | 极端情景下跌破可理解 |

### 三、分红可持续性分析

**步骤1：当前分红水平**
```
分红率 = 年度现金分红总额 / 归母净利润
股息率 = 每股股息 / 股价
```

**步骤2：分红与资本补充的平衡**
```
内生资本补充 = 净利润 × (1 - 分红率) / 风险加权资产
RWA增速 = 风险加权资产同比增长率

可持续分红的条件：
内生资本补充率 > RWA增速 × CET1目标比率
即：银行留存利润产生的资本增量 > 业务扩张消耗的资本
```

**步骤3：压力情景下分红评估**

| 情景 | 判断标准 | 结论 |
|------|---------|------|
| 正常情景 | 分红后CET1缓冲 > 2% | 可持续 |
| 轻度压力 | 分红后CET1缓冲 > 1% | 可持续但需监测 |
| 重度压力 | 分红后CET1仍达标 | 可维持但可能需削减 |
| 极端压力 | 分红后CET1跌破底线 | 必须停止或大幅削减分红 |

**历史分红稳定性**：
- 连续5年以上稳定分红 → 强分红意愿
- 分红率波动大或有削减记录 → 分红可靠性打折
- 国有大行分红率通常30%，近年有提升至35%+的趋势

### 四、资本补充渠道评估

**内源性资本补充**：
- 利润留存：最健康的方式，看留存利润能否覆盖RWA增长
- 超额拨备释放：高拨备覆盖率的银行有"隐藏利润"

**外源性资本补充**：
- 定增/配股：稀释现有股东，最后手段
- 可转债：转股前为债务，转股后补充核心资本
- 永续债：补充其他一级资本
- 二级资本债：补充二级资本

**关键判断**：
- 银行是否在近期（1-2年）有外源融资计划？→ 说明资本内生补充不足
- 可转债是否已进入转股期？转股概率？→ 影响EPS和PB
- 永续债、二级资本债到期续发压力？→ 影响融资成本

### 五、房地产敞口专项压力测试

当前A股银行最核心的尾部风险来源是房地产相关敞口。

**需评估的敞口**：
```
1. 对公房地产贷款：直接的房企贷款
2. 个人住房按揭贷款：居民断供风险
3. 以房地产为抵押的其他贷款：抵押物贬值风险
4. 非标投资中的地产项目：表外风险
5. 地方政府平台贷款中土地相关部分
```

**压力情景**：
- 房价下跌20%：按揭贷款LTV突破预警线，但多数仍有安全垫
- 房价下跌30%：部分城市按揭可能出现负资产，对公地产贷款大面积违约
- 关注银行房地产敞口占总贷款比例，>20%需特别警惕
