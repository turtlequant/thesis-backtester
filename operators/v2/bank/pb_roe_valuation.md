## PB-ROE 估值模型

银行股的核心估值框架是 PB-ROE 模型。净利润受拨备和减值影响波动较大，PE 估值不稳定；而净资产是银行经营的根基，PB 估值更具经济意义。

### 一、PB-ROE 理论基础

**核心公式**：

```
合理PB = (ROE - g) / (COE - g)

其中：
- ROE = 净资产收益率（使用加权平均ROE）
- g = 可持续增长率 ≈ ROE × 留存比例
- COE = 股权成本（Cost of Equity），A股银行一般假设 8%-10%
```

**简化判断**：
- 当 ROE > COE 时，合理PB > 1（值得溢价）
- 当 ROE < COE 时，合理PB < 1（破净合理）
- 当 ROE = COE 时，合理PB = 1

### 二、当前PB诊断

**步骤1：获取当前估值数据**
- 从 price_history 获取最新收盘价，计算当前市值
- 从 balancesheet 获取最新归母净资产
- 计算当前 PB = 市值 / 归母净资产

**步骤2：历史PB分位数**
- 取最近5年PB数据，计算当前PB处于历史百分位
- PB处于历史10%以下分位 → 极端低估信号
- PB处于历史25%-75%分位 → 正常波动区间
- PB处于历史90%以上分位 → 需要警惕高估

**步骤3：同业PB比较**
- 与国有大行、股份行、城商行、农商行对标
- 同类型银行PB差异通常反映ROE差异和资产质量差异

### 三、ROE分析与趋势

**步骤1：ROE计算**
- 使用加权平均ROE（不是简单ROE）
- 从 fina_indicator 获取近5年ROE数据
- 观察ROE趋势：稳定、下滑还是回升

**步骤2：ROE杜邦分解**
```
ROE = 净利润率 × 资产周转率 × 权益乘数
    = (净利润/营收) × (营收/总资产) × (总资产/净资产)

银行简化版：
ROE ≈ ROA × 权益乘数
ROA = 净息差(NIM) × 生息资产占比 - 信用成本 - 运营成本率 + 非息收入率
```

**步骤3：ROE驱动因素诊断**
- NIM下降 → ROA下降 → ROE下降（当前A股银行主要矛盾）
- 信用成本上升 → ROA下降（资产质量恶化信号）
- 权益乘数下降 → ROE下降（监管要求提高资本充足率）

### 四、隐含ROE与合理PB推算

**反向推算**：当前PB隐含市场对ROE的预期
```
implied_ROE = PB × COE（简化公式，假设g=0）
精确版：implied_ROE = PB × (COE - g) + g
```

**正向推算**：基于当前ROE计算合理PB
```
target_PB = (ROE - g) / (COE - g)

参数选取：
- COE = 9%（A股银行中枢假设）
- g = ROE × (1 - 分红率)
- ROE = 使用未来3年预期ROE的中枢值
```

**PB-ROE 匹配度判断**：

| 情形 | 判断 | 说明 |
|------|------|------|
| 当前PB < target_PB × 0.8 | 低估 | 市场给予过多折扣 |
| target_PB × 0.8 ≤ 当前PB ≤ target_PB × 1.2 | 合理 | 估值反映基本面 |
| 当前PB > target_PB × 1.2 | 高估 | 溢价过高 |

### 五、PB < 1 的鉴别诊断

银行PB破净（<1）是否合理需要具体分析：

**PB < 1 合理的情形**：
- ROE持续低于COE（如ROE < 8%），净资产无法创造超额回报
- 存在大量未暴露的不良资产，真实净资产可能低于账面
- 行业系统性风险上升（如房地产敞口过大）

**PB < 1 可能是买入机会**：
- ROE仍维持在10%以上但PB < 0.6 → 市场过度悲观
- 不良率已见顶回落、拨备覆盖率充足 → 资产质量拐点
- 股息率 > 6%且分红可持续 → 高股息保护

**关键验证**：
- 检查ROE是否有回升趋势
- 检查不良率是否已见顶
- 检查拨备覆盖率是否充足（>150%）
- 检查资本充足率是否满足监管要求

### 六、回归分析方法（同业比较）

**操作步骤**：
1. 收集A股上市银行（至少15家）的当前PB和ROE数据
2. 以ROE为X轴、PB为Y轴做散点图
3. 拟合回归线：PB = a × ROE + b
4. 目标银行在回归线上方 → 相对高估；下方 → 相对低估
5. 偏离回归线的程度（残差大小）反映市场给予的额外溢价/折价

**溢价/折价的合理原因**：
- 资产质量优异（不良率低）→ 可享受溢价
- 区域经济发达（如江浙城商行）→ 可享受溢价
- 股东背景强（国有大行）→ 系统重要性溢价
- 治理结构存疑 → 合理折价
- 区域经济风险集中（如东北城商行）→ 合理折价
