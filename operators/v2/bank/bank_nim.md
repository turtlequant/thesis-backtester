## 净息差与盈利能力分析

净息差（NIM）是银行最核心的盈利指标，直接决定息差收入——占多数A股银行总收入的70%-80%。NIM的变化趋势是判断银行利润增长的第一信号。

### 一、净息差（NIM）计算与分析

**定义与公式**：
```
净息差(NIM) = (利息收入 - 利息支出) / 平均生息资产 × 100%
净利差(NIS) = 生息资产收益率 - 计息负债成本率
```

**步骤1：获取NIM数据**
- 从 fina_indicator 获取近5年NIM数据
- 关注NIM的绝对水平和变动趋势

**NIM水平判断**：

| NIM水平 | 判断 | 典型代表 |
|---------|------|---------|
| > 2.5% | 高 | 零售型银行（如招行）、部分城农商行 |
| 2.0% - 2.5% | 中等 | 多数股份行 |
| 1.5% - 2.0% | 偏低 | 国有大行、同业业务占比高的银行 |
| < 1.5% | 低 | 盈利压力大，需关注可持续性 |

**步骤2：NIM趋势诊断**
- 连续3个季度以上收窄 → 盈利能力下行趋势确立
- 单季度环比企稳或回升 → 可能见底信号
- 关注每个季度的NIM变化方向，比绝对值更重要

### 二、NIM驱动因素拆解

**资产端收益率分析**：
```
生息资产收益率 = 利息收入 / 平均生息资产
```
- 贷款收益率：受LPR定价影响，是资产端最核心的变量
- 投资收益率：债券投资、同业投资的回报率
- 资产结构：高收益贷款占比越高，整体收益率越高

**负债端成本率分析**：
```
计息负债成本率 = 利息支出 / 平均计息负债
```
- 存款成本率：核心负债成本，活期占比越高成本越低
- 同业负债成本：受市场利率影响，波动较大
- 负债结构：活期存款占比是核心优势指标

**LPR传导效应**：
```
LPR下调对NIM的影响路径：
1. 新发放贷款利率跟随LPR下降 → 资产端收益率下行
2. 存量贷款在重定价日（通常每年1月1日）调整 → 滞后影响
3. 存款利率跟随下调幅度通常小于LPR降幅 → 负债端粘性
4. 净效果：NIM承压，幅度取决于重定价结构
```

**关键检查**：
- 贷款占生息资产比例：>55%为贷款驱动型，NIM对LPR更敏感
- 活期存款占比：>50%为负债端优势银行（如招行），NIM更稳定
- 同业负债占比：>30%需关注流动性风险和成本波动

### 三、非利息收入分析

**步骤1：非息收入占比**
```
非息收入占比 = (手续费及佣金净收入 + 其他非息收入) / 营业收入 × 100%
```

| 非息收入占比 | 判断 | 说明 |
|-------------|------|------|
| > 35% | 高 | 收入多元化，对息差依赖低（如招行、宁波） |
| 25% - 35% | 中等 | 较好的业务结构 |
| 15% - 25% | 偏低 | 收入高度依赖息差 |
| < 15% | 低 | 几乎纯息差业务，利润对NIM极其敏感 |

**步骤2：非息收入质量拆解**
- **手续费及佣金收入**：财富管理、信用卡、结算、托管等，可持续性强
- **投资收益/公允价值变动**：债券交易收入，波动大，不可持续
- **汇兑损益**：受汇率波动影响，不可预测
- 优质非息收入 = 手续费及佣金净收入；劣质非息收入 = 投资交易收益

**步骤3：中间业务能力**
- 财富管理AUM增速：反映零售银行竞争力
- 代销基金/保险/理财规模：手续费核心来源
- 信用卡业务：交易量、贷款余额、不良率

### 四、成本控制能力

**成本收入比**：
```
成本收入比 = 业务及管理费 / 营业收入 × 100%
```

| 成本收入比 | 判断 | 说明 |
|-----------|------|------|
| < 28% | 优秀 | 效率极高，多为国有大行 |
| 28% - 33% | 良好 | 多数优质股份行 |
| 33% - 40% | 一般 | 城商行、零售转型中的银行 |
| > 40% | 偏高 | 需关注费用控制，可能在大量投入期 |

**趋势关注**：
- 成本收入比持续下降 → 经营杠杆释放，利好
- 成本收入比上升但伴随非息收入高增 → 战略投入期，可接受
- 成本收入比上升且收入停滞 → 效率恶化，负面

### 五、ROA分解与盈利质量

**ROA是银行盈利能力的终极指标**：
```
ROA = 净利润 / 平均总资产
    ≈ NIM × 生息资产占比
      + 非息收入率
      - 信用成本率
      - 运营成本率
      - 税率影响
```

| ROA水平 | 判断 | 说明 |
|---------|------|------|
| > 1.0% | 优秀 | A股仅少数银行能达到 |
| 0.8% - 1.0% | 良好 | 优质股份行水平 |
| 0.6% - 0.8% | 一般 | 国有大行常见水平 |
| < 0.6% | 偏低 | 盈利能力不足 |

**ROA拆解关注点**：
- NIM贡献下降但ROA稳定 → 非息收入或成本控制在补偿
- NIM稳定但ROA下降 → 信用成本上升（资产质量问题）或费用失控
- NIM和ROA同步下降 → 盈利模式面临挑战
