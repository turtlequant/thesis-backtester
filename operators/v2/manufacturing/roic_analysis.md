## ROIC 与资本效率分析

ROIC（投入资本回报率）是衡量企业资本使用效率的核心指标，对重资产制造业尤为重要。ROIC持续高于WACC意味着企业在创造股东价值；低于WACC则是在毁灭价值。

### 一、ROIC 计算

**标准公式**：
```
ROIC = NOPAT / 投入资本

其中：
NOPAT（税后经营利润）= 营业利润 ×（1 - 有效税率）
    或 = (营业收入 - 营业成本 - 管理费用 - 销售费用 - 研发费用) ×（1 - 有效税率）
    注意：不含财务费用（因为要衡量经营效率，与融资结构无关）

投入资本 = 净营运资本 + 固定资产净值 + 无形资产 + 长期待摊 + 在建工程
        = 总资产 - 超额现金 - 无息流动负债
        或 = 股东权益 + 有息负债 - 超额现金
```

**简化计算（数据受限时）**：
```
ROIC ≈ EBIT ×（1 - 25%）/ （股东权益 + 有息负债 - 货币资金 × 50%）

有息负债 = 短期借款 + 长期借款 + 应付债券 + 一年内到期非流动负债
超额现金 ≈ 货币资金 × 50%（保守估计一半为经营所需）
```

**计算注意事项**：
- 投入资本使用期初期末平均值
- 剔除非经常性损益对营业利润的影响
- 若有大额商誉，分别计算含商誉/不含商誉ROIC

### 二、ROIC 趋势分析（5年维度）

**步骤1：计算近5年ROIC**
- 逐年计算ROIC，观察趋势
- 计算5年平均ROIC和标准差

**步骤2：趋势分类**
| 趋势 | 特征 | 信号 |
|------|------|------|
| 持续上升 | 连续3年以上提升 | 竞争优势在增强 |
| 高位稳定 | 波动<2pp且均值>12% | 具有持久竞争优势 |
| 周期波动 | 大幅波动但均值>WACC | 典型周期股特征，需判断周期位置 |
| 持续下降 | 连续3年以上下滑 | 竞争优势在削弱或行业恶化 |
| 长期低于WACC | 5年均值<8% | 结构性价值毁灭 |

**步骤3：周期行业特殊处理**
- 钢铁/煤炭/化工为强周期行业，单年ROIC波动大
- 必须使用5年（至少3年）平均ROIC进行判断
- 周期顶部ROIC>20%不代表优秀，可能即将回落
- 周期底部ROIC<5%不一定代表差，可能即将修复

### 三、ROIC vs WACC：价值创造判断

**WACC估算**：
```
A股制造业WACC经验值：
├── 低风险行业（公用事业、消费）：7-9%
├── 中风险行业（汽车、家电）：8-10%
├── 高风险行业（钢铁、煤炭、化工）：9-12%
└── 简化处理：多数A股制造业取8-10%

更精确的估算：
WACC = 股权比例 × 股权成本 + 债务比例 × 债务成本 ×（1-税率）
股权成本 ≈ 无风险利率(3%) + Beta × 股权风险溢价(6%) ≈ 9-12%
债务成本 ≈ 贷款利率(4-5%)
```

**价值创造判断**：
| ROIC vs WACC | 判定 | 投资含义 |
|-------------|------|---------|
| ROIC > WACC + 5pp | 显著创造价值 | 优质企业，值得溢价 |
| ROIC > WACC | 创造价值 | 正常企业 |
| ROIC ≈ WACC（差值<1pp） | 持平 | 不创造也不毁灭价值 |
| ROIC < WACC | 毁灭价值 | 长期持有会损失财富 |
| ROIC < WACC - 5pp | 严重毁灭价值 | 除非有明确改善逻辑，否则回避 |

### 四、ROIC 杜邦分解

```
ROIC = NOPAT利润率 × 投入资本周转率

NOPAT利润率 = NOPAT / 营业收入（衡量盈利能力）
投入资本周转率 = 营业收入 / 投入资本（衡量资产使用效率）
```

**分解分析**：
| 组合 | 典型行业 | 改善方向 |
|------|---------|---------|
| 高利润率 + 低周转 | 化工（差异化产品） | 提高产能利用率 |
| 低利润率 + 高周转 | 钢铁（标准化商品） | 成本控制、产品升级 |
| 高利润率 + 高周转 | 优秀企业（少见） | 维持竞争优势 |
| 低利润率 + 低周转 | 陷入困境 | 需要重大变革 |

**动态分析**：
- ROIC提升是靠利润率改善还是周转率提升？
- 利润率改善来源：价格上涨、成本下降、产品结构优化？
- 周转率改善来源：产能利用率提高、库存管理优化、应收账款控制？

### 五、增量ROIC（边际投资回报）

```
增量ROIC = 增量NOPAT / 增量投入资本
        = (本年NOPAT - 去年NOPAT) / (本年投入资本 - 去年投入资本)
```

**关键判断**：
- 增量ROIC > 存量ROIC → 新投资质量更高，企业在改善
- 增量ROIC < 存量ROIC → 新投资回报递减，可能过度扩张
- 增量ROIC < WACC → 新投资在毁灭价值，应停止扩张并返还现金

**警示信号**：
- 大规模CAPEX + 增量ROIC下降 → 管理层可能在"帝国建设"
- 连续3年增量ROIC < WACC → 资本配置能力差

### 六、同业对比

**对比维度**：
- 同行业前5家公司的ROIC排名
- 公司ROIC vs 行业中位数
- 公司ROIC趋势 vs 行业趋势（是公司特有还是行业共性）

**行业ROIC参考水平**：
| 行业 | 优秀 | 中等 | 偏低 |
|------|------|------|------|
| 钢铁 | >12% | 6-12% | <6% |
| 煤炭 | >15% | 8-15% | <8% |
| 化工 | >12% | 7-12% | <7% |
| 汽车 | >15% | 8-15% | <8% |
