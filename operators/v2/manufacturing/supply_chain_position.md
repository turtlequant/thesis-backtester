## 供应链地位分析

供应链地位决定企业在产业链中的议价能力和利润分配。应收/应付账款天数的对比是判断上下游议价能力最直接的财务指标；现金转换周期反映企业对运营资金的占用效率。

### 一、应收账款分析（对下游的议价能力）

**步骤1：计算应收账款周转天数**

```
应收账款周转天数 = 365 / 应收账款周转率
应收账款周转率 = 营业收入 / 平均应收账款余额
平均应收账款余额 = (期初应收 + 期末应收) / 2

注意事项：
├── 应收账款包括应收票据（尤其是商业承兑汇票）
├── 银行承兑汇票信用风险低，可单独列示
├── 关注应收账款与收入的匹配关系
└── 取近5年数据观察趋势
```

**步骤2：应收账款趋势诊断**

| 趋势 | 含义 | 信号 |
|------|------|------|
| 天数持续缩短 | 回款能力增强 | 对下游议价力提升 |
| 天数稳定 | 行业惯例稳定 | 中性 |
| 天数持续延长 | 被迫放宽信用 | 对下游议价力下降 |
| 天数突然跳升 | 可能存在坏账风险 | 警惕大客户风险 |

### 二、应付账款分析（对上游的议价能力）

**步骤1：计算应付账款周转天数**

```
应付账款周转天数 = 365 / 应付账款周转率
应付账款周转率 = 营业成本 / 平均应付账款余额

注意事项：
├── 应付账款包括应付票据
├── 区分商业承兑汇票和银行承兑汇票
├── 过长的应付天数可能表示占用供应商资金
└── 也可能表示资金链紧张被迫拖欠
```

**步骤2：应付账款趋势诊断**

| 趋势 | 含义 | 信号 |
|------|------|------|
| 天数延长且毛利稳定 | 主动占用上游资金 | 对上游议价力强 |
| 天数延长且毛利下降 | 可能是资金紧张 | 需结合现金流判断 |
| 天数缩短 | 上游要求更严格付款 | 对上游议价力减弱 |

### 三、现金转换周期（CCC）

**核心公式**：

```
CCC = 应收账款天数 + 存货周转天数 - 应付账款天数

其中：
├── 存货周转天数 = 365 / (营业成本 / 平均存货)
├── CCC为负 → 先收钱后付款，经营效率极高（如家电龙头）
├── CCC为正 → 需要自行垫资运营
└── CCC越小越好（在不损害供应关系的前提下）
```

**CCC趋势分析**：
- 近5年CCC逐年缩短 → 供应链管理改善，正面信号
- 近5年CCC逐年延长 → 运营效率恶化或议价能力下降
- CCC波动大 → 可能与行业周期、库存策略相关

### 四、收入集中度分析

**步骤1：从fina_mainbz获取分产品/分客户收入数据**

```
分析维度：
├── 前5大客户收入占比
│   ├── > 50% → 客户集中度高，议价被动
│   ├── 30%-50% → 中等集中度
│   └── < 30% → 客户分散，议价主动
│
├── 前5大产品收入占比
│   ├── 单一产品 > 60% → 产品集中度过高
│   └── 多产品均衡 → 抗风险能力强
│
└── 客户集中度趋势
    ├── 集中度上升 → 大客户依赖加深，议价空间被压缩
    └── 集中度下降 → 客户结构优化
```

### 五、毛利率稳定性（定价能力代理变量）

**步骤1：计算近5年毛利率序列**

```
毛利率 = (营业收入 - 营业成本) / 营业收入

定价能力判断：
├── 毛利率标准差 < 2pp → 定价能力强（能转嫁成本波动）
├── 毛利率标准差 2pp-5pp → 定价能力中（部分转嫁）
├── 毛利率标准差 > 5pp → 定价能力弱（被动接受价格）
│
├── 毛利率趋势上升 + 收入增长 → 强定价能力
├── 毛利率趋势下降 + 收入增长 → 以价换量
└── 毛利率趋势下降 + 收入下降 → 量价齐跌，地位恶化
```

**步骤2：与行业对比**
- 毛利率高于行业中位数 → 具备一定定价权
- 毛利率低于行业中位数 → 可能是成本竞争型企业

### 六、上下游地位综合判定

```
议价能力矩阵：

              应付天数长（占上游资金）  应付天数短（被上游压制）
应收天数短    ┌─────────────────────┬─────────────────────┐
（掌控下游）  │  供应链核心企业      │  下游强势但上游也强  │
              │  定价能力：强        │  定价能力：中        │
              ├─────────────────────┼─────────────────────┤
应收天数长    │  上游弱势但下游压款  │  产业链弱势企业      │
（被下游压制）│  定价能力：中        │  定价能力：弱        │
              └─────────────────────┴─────────────────────┘
```
