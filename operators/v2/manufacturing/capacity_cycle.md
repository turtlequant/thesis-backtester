## 产能周期分析

产能周期是钢铁、煤炭、化工等重资产周期行业最重要的分析框架。行业利润不取决于需求增长，而取决于供需缺口。理解产能周期的位置，就能预判行业利润的拐点。

### 一、产能周期四阶段模型

```
扩张期 → 繁荣顶部 → 收缩期 → 出清底部 → 扩张期（循环）

扩张期（CAPEX/折旧 > 1.5x）：
├── 特征：行业利润高 → 企业大量投资新产能
├── 信号：在建工程大幅增长，CAPEX加速
├── 陷阱：利润最好的时候往往是周期顶部前夜
└── 投资策略：回避或减仓，利润即将见顶

繁荣顶部（产能集中释放）：
├── 特征：新产能集中投产，供给快速增加
├── 信号：收入增长放缓但固定资产仍在增加
├── 价格：产品价格开始松动，毛利率见顶回落
└── 投资策略：坚决回避

收缩期（CAPEX/折旧 < 1.0x）：
├── 特征：行业亏损 → 企业停止投资，减少资本开支
├── 信号：CAPEX大幅缩减，部分企业开始关停产能
├── 关键：高成本产能是否在退出市场
└── 投资策略：开始关注，等待出清信号

出清底部（供需再平衡）：
├── 特征：落后产能退出，供给收缩到需求以下
├── 信号：产品价格企稳回升，龙头企业利润率改善
├── 确认：行业CAPEX连续2-3年低于折旧
└── 投资策略：最佳买入时机
```

### 二、CAPEX/折旧比率分析

**计算方法**：
```
CAPEX = 购建固定资产、无形资产和其他长期资产支付的现金
折旧 = 固定资产折旧 + 无形资产摊销 + 长期待摊费用摊销
CAPEX/折旧比率 = CAPEX / 折旧
```

**判断标准**：
| CAPEX/折旧 | 含义 | 产能阶段 |
|-----------|------|---------|
| > 2.0x | 激进扩张 | 扩张期高峰 |
| 1.5-2.0x | 积极扩张 | 扩张期 |
| 1.0-1.5x | 温和扩张/维持 | 成熟期 |
| 0.8-1.0x | 仅维持性投入 | 开始收缩 |
| < 0.8x | 产能收缩 | 收缩期/出清期 |
| < 0.5x | 严重缩减投入 | 深度出清 |

**趋势分析（必须看3-5年）**：
- 连续3年CAPEX/折旧 > 1.5x → 行业处于扩张期
- CAPEX/折旧从>2x降至<1x → 周期从扩张转向收缩
- 连续2年CAPEX/折旧 < 0.8x → 产能出清进行中

### 三、在建工程与固定资产增长率

**在建工程分析**：
```
在建工程变化 = (期末在建工程 - 期初在建工程) / 期初在建工程

解读：
├── 在建工程大幅增长(>50%) → 未来1-2年将有大量新产能投放
├── 在建工程持平 → 扩张减速
├── 在建工程大幅减少(>30%) → 新投资基本停止
└── 在建工程转固定资产 → 产能正在释放
```

**固定资产增长率**：
```
固定资产增长率 = (期末固定资产 - 期初固定资产) / 期初固定资产

注意：固定资产增长 = 在建转固(新产能) - 折旧(老产能消耗) + 收购
├── 增长率 > 折旧率 → 净产能在扩张
├── 增长率 ≈ 折旧率 → 净产能维持
└── 增长率 < 折旧率 → 净产能在收缩
```

### 四、产能利用率估算

直接产能利用率数据较难获取，可通过以下间接指标估算：

**方法1：收入/固定资产比率（资产周转率代理）**
```
单位固定资产产出 = 营业收入 / 固定资产净值

趋势分析：
├── 上升 → 产能利用率在提高（量升或价升）
├── 下降 → 产能利用率在降低或新产能未达产
└── 需剔除价格因素影响（用量的指标更佳）
```

**方法2：毛利率反推**
```
周期行业毛利率与产能利用率高度相关：
├── 毛利率处于历史高位 → 产能利用率高，供不应求
├── 毛利率处于历史低位 → 产能利用率低，供过于求
└── 毛利率拐点往往对应产能利用率拐点
```

**方法3：存货周转天数**
```
存货周转天数上升 → 产品销售放缓 → 产能过剩信号
存货周转天数下降 → 产品供不应求 → 产能紧张信号
```

**综合判断**：
| 利用率估计 | 判断依据 | 信号 |
|-----------|---------|------|
| 高（>85%） | 毛利率高位+存货低+周转快 | 供给紧张 |
| 中（70-85%） | 指标处于历史中位 | 供需平衡 |
| 低（<70%） | 毛利率低位+存货高+周转慢 | 产能过剩 |

### 五、行业供需平衡信号

**供给侧信号**：
```
供给收缩信号（正面）：
├── 行业CAPEX连续2年下降
├── 环保/安全政策导致落后产能关停
├── 行业内企业数量减少（兼并重组/破产退出）
├── 龙头企业市占率提升
└── 进口替代减少（国内供给收缩）

供给扩张信号（负面）：
├── 多家企业同时宣布扩产计划
├── 在建工程行业汇总大幅增长
├── 新进入者增多
└── 海外产能转移至国内
```

**需求侧信号**：
```
需求向好：
├── 下游行业景气度上升
├── 出口订单增长
├── 基建/地产投资加速
└── 新兴需求领域出现（如新能源对化工品的需求）

需求走弱：
├── 下游去库存
├── 出口订单下降
├── 基建/地产投资放缓
└── 替代品冲击
```

### 六、周期位置综合判断矩阵

| 指标 | 底部 | 复苏 | 繁荣 | 衰退 |
|------|------|------|------|------|
| CAPEX/折旧 | <0.8x | 0.8-1.2x | >1.5x | 1.0-1.5x(减速) |
| 在建工程 | 大幅减少 | 持平/微增 | 大幅增长 | 开始减少 |
| 毛利率 | 历史低位 | 回升中 | 历史高位 | 回落中 |
| 存货 | 低位 | 正常 | 开始积累 | 高位 |
| 产品价格 | 企稳 | 上涨 | 高位 | 回落 |
| 行业情绪 | 极度悲观 | 谨慎乐观 | 极度乐观 | 开始担忧 |

**投资决策指引**：
- 底部+出清信号明确 → 最佳买入时机（逆向投资）
- 复苏初期 → 仍可买入，但仓位适当控制
- 繁荣期 → 回避或减仓，不追高周期股
- 衰退期 → 耐心等待，不要抄底过早
