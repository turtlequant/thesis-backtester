## 管理人诚信与治理风险评估

本算子用于系统评估上市公司管理层的诚信度和公司治理质量。管理人诚信是价值投资的前提条件 -- 再便宜的公司，如果管理层不可信，投资安全边际也为零。

### 数据回退方案

本算子涉及多维度治理评估，部分检查项需要年报文本、公告原文等非结构化数据，当前数据层不完全覆盖。以下为各维度的数据可用性说明：

| 检查维度 | 可用数据 | 不可用数据 | 回退处理 |
|----------|----------|-----------|----------|
| 承诺兑现 | dividend（分红记录） | 增持/回购公告原文、业绩指引 | 仅验证分红一致性，其余标注"无法验证" |
| 关联交易 | balancesheet（应收/其他应收）、income | 关联交易明细、定价公允性 | 通过其他应收款异常增长间接识别，无法直接验证 |
| 信息披露 | fina_audit（审计意见） | 年报详细度对比、会计政策变更 | 仅检查审计意见类型，其余标注"无法验证" |
| 股权行为 | stk_holdertrade（增减持）、pledge_stat（质押统计）、top10_holders | 一致行动人代持安排 | 可检查减持记录和质押比例，代持标注"无法验证" |
| CEO/董事长变更 | basic_info（现任信息） | 历史变更记录 | 仅能看到当前管理层，频繁变更标注"无法验证" |

**重要原则**：对于数据不可用的检查项，必须在输出中标注"无法验证（数据不可用）"，而非跳过或假设通过。

### 一、评估维度与检查项

#### 维度1：承诺兑现与分红一致性

```
检查清单：
  [ ] 分红政策是否稳定执行？（数据来源：dividend）
      - 连续5年以上稳定分红 -> 正面信号
      - 派息率是否与公开政策一致
      - 分红金额是否与盈利能力匹配（避免借债分红或有利润不分红）
  [ ] 过去3年公开增持承诺是否全部执行？-> 无法验证（需公告原文）
  [ ] 回购计划是否按期完成？-> 无法验证（需公告原文）
  [ ] 业绩指引/展望的准确度？-> 无法验证（需公告原文）

评分（基于可验证项）：
  分红连续稳定、派息率合理 -> 正面
  分红不稳定但有合理解释（周期性行业等） -> 中性
  有利润不分红或借债分红 -> 负面
```

#### 维度2：关联交易与利益输送

```
检查清单（财报可识别）：
  [ ] 其他应收款/其他应付款异常？（数据来源：balancesheet）
      其他应收款 > 营收的20% -> 警惕资金被占用
      其他应收款连续增长 -> 高风险
  [ ] 大存大贷？（数据来源：balancesheet）
      货币资金 > 总资产的20% 但同时有息负债也 > 总资产的20% -> 异常
  [ ] 应收账款异常增长？（数据来源：balancesheet, income）
      应收账款增速 > 营收增速的2倍 -> 关注收入质量
  [ ] 经营现金流与净利润严重背离？（数据来源：cashflow, income）
      连续3年 经营现金流/净利润 < 0.5 -> 利润质量可疑

无法验证项（标注即可）：
  [ ] 关联交易总额占营收比例 -> 无法验证（需关联交易明细）
  [ ] 关联交易定价公允性 -> 无法验证（需交易明细）
  [ ] 担保结构 -> 无法验证（需担保明细公告）
```

**重点关注的资金操纵手法**：
| 手法 | 财报识别特征 | 数据来源 | 风险等级 |
|------|-------------|----------|----------|
| 大存大贷 | 高额货币资金+高额有息负债并存 | balancesheet | 高 |
| 资金占用 | 其他应收款异常且持续增长 | balancesheet | 极高 |
| 收入造假 | 应收增速远超营收增速+现金流背离 | balancesheet, income, cashflow | 极高 |
| 费用资本化 | 在建工程/开发支出异常增长 | balancesheet | 高 |

#### 维度3：信息披露质量

```
检查清单：
  [ ] 审计意见类型？（数据来源：fina_audit）
      标准无保留 -> 正常
      带强调事项段 -> 需关注
      保留意见/否定意见 -> 高风险
      无法获取/无法发表意见 -> 极高风险
  [ ] 是否更换过审计师？-> 若 fina_audit 有历史记录则检查，否则标注"无法验证"

无法验证项：
  [ ] 年报篇幅与详细程度 -> 无法验证（需年报原文）
  [ ] 风险因素披露是否具体 -> 无法验证
  [ ] 会计政策是否频繁变更 -> 无法验证
  [ ] 内控报告重大缺陷 -> 无法验证
```

#### 维度4：股权行为与稀释风险

```
检查清单：
  [ ] 大股东/高管减持记录？（数据来源：stk_holdertrade）
      股价低位减持 -> 极度负面信号
      大额减持（占持股比例 > 10%） -> 负面信号
  [ ] CEO/董事长变更频繁？（数据来源：basic_info 当前管理层信息）
      3年内董事长或CEO变更2次以上 -> 治理不稳定信号
      注意：仅能看到当前管理层，历史变更可能无法验证
  [ ] 股权质押比例？（数据来源：pledge_stat）
      > 50% -> 高风险（平仓压力）
      > 80% -> 极高风险
  [ ] 定向增发（定增）稀释风险？（数据来源：top10_holders 持股比例变化）
      定增折价率超过20%且频繁实施（3年内2次以上） -> 稀释风险
      持续定增但项目回报不及预期 -> 管理层资本配置能力存疑
  [ ] 大股东持股比例变化趋势？（数据来源：top10_holders）
      持续下降 -> 关注原因

无法验证项：
  [ ] 一致行动人代持安排 -> 无法验证
  [ ] 期权行权后立即抛售 -> 无法验证（需期权明细）
```

### 二、特殊红旗信号（一票否决项）

以下任一项出现即应高度警惕或直接回避：

```
[!] 审计师出具否定意见或无法发表意见（fina_audit 可验证）
[!] 大股东占用上市公司资金——其他应收款异常（balancesheet 可间接识别）
[!] 频繁更换CFO/CEO/董事长（3年内2次以上）（basic_info 部分可验证）
[!] 连续定增稀释小股东且项目回报差（top10_holders + fina_indicator 可间接验证）
[!] 股权质押比例 > 80%（pledge_stat 可验证）
[!] 经营现金流连续3年为负但持续分红（cashflow + dividend 可验证）

以下为高风险信号但当前数据无法直接验证，如有外部信息来源可补充：
[!] 管理层因财务造假被监管处罚 -> 无法验证
[!] 大股东占用资金（经监管认定） -> 无法验证（仅可从财报间接推断）
[!] 实际控制人涉及刑事案件 -> 无法验证
```

### 三、三级评级体系

#### 可信（投资前提满足）

- 分红连续稳定（5年以上），派息率合理
- 无大存大贷、无其他应收款异常
- 标准无保留审计意见
- 无大额减持记录、无频繁定增
- 质押比例 < 30% 或无质押
- 无一票否决项触发
- CEO/董事长任期稳定

#### 存疑（需额外安全边际补偿）

- 分红不稳定但有行业合理性
- 存在轻微大存大贷但金额不极端
- 有少量减持但时机和规模尚可
- 质押比例 30%-50%
- 部分检查项"无法验证"
- **处置方式**：若投资，需在估值上要求额外20%-30%的安全边际

#### 不可信（建议回避）

- 一票否决项触发任一条
- 大存大贷严重、其他应收款持续异常增长
- 审计意见非标
- 大股东在股价低位大量减持 + 高比例质押（>50%）
- 连续定增但项目回报差
- 经营现金流长期背离净利润
- **处置方式**：无论估值多便宜，均不投资

### 四、分析输出模板

1. **分红一致性**：连续分红[N]年，派息率[__]%，[正面/中性/负面]
2. **资金安全**：其他应收款占营收[__]%，大存大贷[有/无]，现金流/利润比[__]，[正常/关注/高风险]
3. **信息披露**：审计意见[类型]，审计师变更[有/无/无法验证]
4. **股权行为**：大股东减持[有/无]，定增历史[有/无]，质押比例[__]%，CEO/董事长变更[稳定/频繁/无法验证]
5. **红旗信号**：[有/无]一票否决项，具体为[...]
6. **数据局限**：以下检查项无法验证：[列出]
7. **综合评级**：[可信/存疑/不可信]
8. **投资建议**：基于诚信评估的处置建议
