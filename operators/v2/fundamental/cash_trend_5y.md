## 5年现金余额与自由现金流趋势分析

本算子指导完成对目标公司近5年现金及现金流的全面趋势分析，评估利润质量、现金创造能力和财务安全边际，并检测现金流拐点信号。

### 一、数据采集框架

从最近5个完整财年提取以下数据，填入下表：

| 指标 | Y-4 | Y-3 | Y-2 | Y-1 | Y0 | 趋势 |
|------|-----|-----|-----|-----|-----|------|
| 货币资金（期末） | | | | | | |
| 交易性金融资产 | | | | | | |
| **广义现金合计** | | | | | | |
| 经营活动现金流净额 | | | | | | |
| 资本支出（购建固定资产等） | | | | | | |
| **自由现金流(FCF)** | | | | | | |
| 净利润 | | | | | | |
| 投资活动现金流净额 | | | | | | |
| 筹资活动现金流净额 | | | | | | |

广义现金 = 货币资金 + 交易性金融资产 + 其他短期理财
自由现金流(FCF) = 经营活动现金流净额 - 资本支出

### 二、核心比率计算

#### 1. 现金转化率（利润质量核心指标）

```
现金转化率 = 经营活动现金流净额 / 净利润

评级标准：
  > 120%  : 优秀 -- 利润含金量极高，可能有折旧等非现金回收
  100%-120%: 良好 -- 利润基本由现金支撑
  70%-100% : 一般 -- 部分利润为应收账款等非现金项
  50%-70%  : 较差 -- 利润质量可疑，需检查应收增长
  < 50%    : 危险 -- 利润严重依赖非现金项，可能存在造假风险
```

#### 2. 自由现金流收益率

```
FCF收益率 = 自由现金流 / 总市值（阈值已与策略统一标准对齐）

  > 12%  : 极度低估（通常伴随市场恐慌或结构性折价）
  8%-12% : 显著低估
  5%-8%  : 合理偏低
  3%-5%  : 合理
  < 3%   : 偏贵
  < 0%   : 需分析原因（扩产期 vs 经营恶化）
```

#### 3. 现金覆盖月数

```
现金覆盖月数 = 广义现金 / (月均经营支出)
月均经营支出 = (营业成本 + 管理费用 + 销售费用) / 12

  > 12个月 : 安全边际充足
  6-12个月 : 正常
  3-6个月  : 偏紧，需关注
  < 3个月  : 高风险，流动性危机可能
```

### 三、趋势分析决策树

```
5年广义现金趋势？
  |
  +-- 持续增长 -> 正面信号
  |     +-- 增长来源？
  |           +-- 经营现金流驱动 -> 优秀（内生增长）
  |           +-- 融资现金流驱动 -> 需警惕（借钱堆现金）
  |           +-- 投资回收驱动 -> 中性（关注是否卖核心资产）
  |
  +-- 基本稳定 -> 中性
  |     +-- 每年FCF为正 -> 现金在循环使用，可接受
  |     +-- 每年FCF为负 -> 持续消耗，需检查投资回报
  |
  +-- 持续下降 -> 警告信号
        +-- 连续3年下降 -> 高度警惕
        +-- 伴随负债增加 -> 双重恶化
        +-- 伴随利润增长 -> 利润质量极差（纸面利润）
```

### 四、现金流组合模式诊断

| 经营 | 投资 | 筹资 | 模式名称 | 含义 |
|------|------|------|----------|------|
| + | - | + | 扩张期 | 经营赚钱+借钱投资，高速增长 |
| + | - | - | 成熟优质 | 经营赚钱+投资+还债，最健康 |
| + | + | - | 收缩转型 | 经营赚钱+卖资产+还债，去杠杆 |
| + | + | + | 囤积现金 | 罕见，需排查目的 |
| - | + | + | 危机模式 | 经营亏损+卖资产+借钱续命 |
| - | - | + | 烧钱扩张 | 亏损+投资+靠融资，初创期或危机 |
| - | + | - | 衰退清算 | 经营失血+卖资产还债 |
| - | - | - | 全面恶化 | 各方面都在失血 |

### 五、警示信号清单

```
[ ] 现金余额连续3年下降
[ ] FCF连续3年为负且非合理扩产期
[ ] 现金转化率 < 50%（连续2年）
[ ] 净利润增长但经营现金流下降（剪刀差扩大）
[ ] 大量现金但同时高额借款（大存大贷）
[ ] 现金覆盖月数 < 6个月且无充足授信
[ ] 应收账款增速 > 营收增速（赊销堆量）
[ ] 经营现金流高度依赖单一大客户回款
```

触发3项及以上 -> 现金流质量"差"
触发1-2项 -> 现金流质量"需关注"
无触发 -> 现金流质量"良好"

### 六、周期拐点现金流特征

当分析周期性行业时，以下现金流组合暗示底部反转：

```
拐点信号检测：
  [ ] 经营现金流同比增长 > +30%
  [ ] 投资现金流由负转正（处置资产回收现金）
  [ ] 筹资现金流为负（主动偿还债务）
  [ ] FCF由负转正
  [ ] 应付账款增加（恢复对供应商的正常信用期）

满足3项以上 -> 现金流层面确认拐点
```
