## 负债结构深度拆解

本算子指导完成对目标公司负债结构的全面拆解，区分有息负债与经营性负债，识别隐性负债风险，并结合行业特征给出负债质量评级。

### 一、负债分类树

```
总负债
  |
  +-- 有息负债（核心风险来源）
  |     +-- 短期借款（1年内到期，流动性压力最大）
  |     +-- 长期借款（银行贷款，关注到期时间表）
  |     +-- 应付债券/票据（公开市场融资，关注利率与到期分布）
  |     +-- 租赁负债（IFRS16/新租赁准则下的融资性质负债）
  |     +-- 其他有息项（信托贷款、委托贷款、资产支持证券等）
  |
  +-- 经营性负债（业务自然产生，通常低风险）
  |     +-- 应付账款（对上游供应商的欠款 -> 反映议价能力）
  |     +-- 预收款项/合约负债（客户预付 -> 正面信号）
  |     +-- 应付职工薪酬（工资社保等）
  |     +-- 应交税费（税务负债）
  |     +-- 其他应付款（需逐项检查是否含隐性有息负债）
  |
  +-- 特殊项目（需单独判断）
        +-- 递延所得税负债（非现金，一般可忽略）
        +-- 递延收益（政府补助等，通常正面）
        +-- 预计负债（诉讼、担保等或有负债）
```

### 二、"零有息负债"快速识别法（30秒规则）

**步骤1：看利润表财务费用**
- 财务费用 < 0（为负数）: 强烈信号，公司利息收入 > 利息支出，大概率无有息负债
- 财务费用接近0: 可能有少量有息负债，需进一步核实

**步骤2：交叉验证（年报附注）**
- 查看利息支出明细：利息支出 < 利息收入 -> 确认无有息负债
- 查看银行借款明细：期末余额为0 -> 确认

**步骤3：汇总确认**
- 同时满足"财务费用为负"+"银行借款为0"+"无应付债券" -> 判定为无有息负债

### 三、核心比率计算

| 指标 | 公式 | 健康标准 | 警戒线 |
|------|------|----------|--------|
| 有息负债率 | 有息负债 / 总负债 | < 30% | > 60% |
| 资产负债率 | 总负债 / 总资产 | 行业相关 | 见下表 |
| 净负债率 | (有息负债 - 现金) / 净资产 | < 50% | > 100% |
| 短期偿债比 | 短期有息负债 / 货币资金 | < 1.0 | > 2.0 |
| 利息覆盖倍数 | EBITDA / 利息支出 | > 5x | < 2x |

### 四、行业资产负债率容忍度参考

| 行业类型 | 合理区间 | 特殊说明 |
|----------|----------|----------|
| 轻资产服务（物业/软件/咨询） | 30%-50% | 高于50%需警惕 |
| 消费品制造 | 40%-60% | 关注应付账款占比 |
| 重资产制造（化工/钢铁） | 50%-70% | 周期底部可放宽至75% |
| 基础设施/公用事业 | 60%-75% | 稳定现金流支撑 |
| 银行/金融 | 90%+ | 行业特殊性，不适用此框架 |
| 房地产 | 70%-80% | 关注净负债率而非资产负债率 |

### 五、隐性负债排查清单

```
[ ] "其他应付款"是否包含借款性质款项？（关联方借款常藏于此）
[ ] 表外担保总额占净资产比例？（> 50%为高风险）
[ ] 是否存在"大存大贷"？（高现金+高借款并存，资金可能受限）
    判定标准：货币资金 > 总负债30%，同时有息负债 > 净资产50%
[ ] 应收票据是否已背书/贴现但未终止确认？
[ ] 合并报表外是否有重大投资或担保？
[ ] 永续债是否计入权益而实质为债务？
    判定：票面利率跳升条款 + 发行人无权递延付息 -> 实质为债
```

### 六、周期底部债务容忍度

> 周期底部的债务容忍度评估由 `cycle_analysis` 算子负责，此处不重复。本算子专注于债务结构的静态分解。
> 如需评估周期底部的债务安全性，请参考 cycle_analysis 的相关输出。

### 七、分析输出模板

完成以上分析后，按以下格式输出结论：

1. **有息负债绝对额与占比**：[数值]，[占总负债比例]
2. **经营性负债质量**：应付账款/合约负债是否反映议价能力
3. **偿债压力评估**：短期到期有息负债 vs 可用现金
4. **隐性负债风险**：是否发现表外担保/大存大贷等异常
5. **周期容忍判定**（如适用）：参见 cycle_analysis 算子输出
6. **负债质量评级**：优（无有息负债）/ 良（有息负债可控）/ 差（高有息+短期集中到期）/ 危（资不抵债风险）
