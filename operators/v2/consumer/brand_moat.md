## 品牌护城河分析

消费品企业的核心竞争优势往往体现为品牌护城河。品牌护城河的本质是消费者心智中的定价权——能够长期维持超额毛利率而不被竞争侵蚀。本算子通过财务数据定量评估品牌护城河的强度和持续性。

### 一、毛利率趋势分析（品牌强度的财务指纹）

**步骤1：计算近5年毛利率序列**

```
毛利率 = (营业收入 - 营业成本) / 营业收入

从 income 表提取近5年年度数据，计算逐年毛利率。

品牌强度判定：
├── 毛利率 > 60% 且稳定上升 → 顶级品牌（如高端白酒）
├── 毛利率 40%-60% 且稳定  → 强品牌
├── 毛利率 30%-40% 且稳定  → 中等品牌
├── 毛利率 < 30% 或持续下降 → 品牌力弱，靠成本竞争
```

**步骤2：毛利率稳定性评估**

```
计算5年毛利率标准差：
├── 标准差 < 1.5pp → 极度稳定，品牌溢价牢固
├── 标准差 1.5pp-3pp → 较稳定，轻微波动可接受
├── 标准差 3pp-5pp → 不太稳定，品牌力受竞争侵蚀
└── 标准差 > 5pp → 不稳定，缺乏真正的品牌护城河
```

**步骤3：毛利率趋势方向**

| 趋势 | 含义 | 品牌信号 |
|------|------|---------|
| 持续上升（3年+） | 品牌升级或产品结构优化 | 品牌力增强 |
| 高位稳定 | 品牌溢价维持 | 品牌力坚固 |
| 缓慢下降 | 竞争加剧或被迫让利 | 品牌力被侵蚀 |
| 快速下降 | 品牌失势或价格战 | 护城河坍塌 |

### 二、提价能力分析

**步骤1：量价拆分**

```
从 fina_indicator 和 income 获取：
├── 营业收入增速 = 名义增长（量 × 价）
├── 如有分产品数据(fina_mainbz)可进一步拆分
│
├── 收入增速 > 行业量增 → 存在提价贡献或份额提升
├── 收入增速 ≈ 行业量增 → 主要靠量驱动，提价有限
└── 收入增速 < 行业量增 → 可能在降价保量
```

**步骤2：提价能力证据链**

```
强提价能力的特征（需同时满足多条）：
├── 毛利率稳定或上升（能将成本上涨转嫁）
├── 收入增速持续高于行业均值
├── 销售费用率稳定或下降（品牌自带流量）
├── 经营现金流/净利润 > 1（下游付款意愿强）
│
弱提价能力的特征：
├── 原材料涨价时毛利率同步下降
├── 销售费用率持续上升（花钱买增长）
├── 应收账款增速 > 收入增速（被迫放宽信用）
└── 频繁促销或降价（从渠道端数据可验证）
```

**步骤3：提价能力评级**

| 等级 | 标准 |
|------|------|
| 强 | 毛利率上升或稳定 + 收入增速高 + 销售费用率下降 |
| 中 | 毛利率小幅波动 + 收入增速中等 + 销售费用率稳定 |
| 弱 | 毛利率下降 + 收入增速靠量 + 销售费用率上升 |

### 三、产品收入集中度

**步骤1：从fina_mainbz获取分产品收入数据**

```
分析维度：
├── 核心产品收入占比
│   ├── 单一产品 > 70% → 高集中度（大单品模式）
│   │   ├── 优势：聚焦、规模效应、品牌认知集中
│   │   └── 风险：产品生命周期、消费偏好变迁
│   │
│   ├── 前3产品 50%-70% → 适中（多品类协同）
│   └── 前3产品 < 50% → 分散（多元化经营）
│
├── 新产品贡献率
│   ├── 新品收入占比上升 → 创新能力强，品牌延展力好
│   └── 新品失败率高 → 品牌延展力有限
│
└── 产品结构变化趋势
    ├── 高毛利产品占比上升 → 品牌升级成功
    └── 低毛利产品占比上升 → 可能在降维竞争
```

### 四、竞争格局与市场份额稳定性

**步骤1：行业地位判断**

```
通过以下指标间接判断市场份额稳定性：
├── 收入规模在行业中的排名是否稳定
├── 收入增速 vs 行业增速
│   ├── 持续高于行业 → 市场份额提升
│   ├── 与行业持平 → 份额稳定
│   └── 低于行业 → 份额流失
│
├── 销售费用率趋势
│   ├── 份额稳定 + 费用率下降 → 品牌自然获客（最优）
│   ├── 份额提升 + 费用率上升 → 花钱买份额（需评估ROI）
│   └── 份额下降 + 费用率上升 → 竞争恶化（危险）
│
└── 竞争格局判定
    ├── CR3 > 60% + 自身是Top3 → 格局清晰，份额稳定
    ├── CR3 30%-60% → 竞争中等，需持续投入
    └── CR3 < 30% → 高度分散，品牌护城河有限
```

### 五、无形资产价值

**步骤1：品牌资产的财务间接评估**

```
品牌资产 ≈ 超额盈利能力的持续性

估算方法：
├── 超额毛利率 = 企业毛利率 - 行业中位数毛利率
├── 品牌贡献利润 ≈ 超额毛利率 × 营业收入
├── 品牌资产倍数 = 品牌贡献利润 × 合理倍数（8-15x）
│
├── 超额毛利率 > 15pp → 强品牌资产
├── 超额毛利率 5pp-15pp → 中等品牌资产
└── 超额毛利率 < 5pp → 品牌资产有限
```

### 六、护城河综合评估框架

```
护城河评分矩阵（5个维度各1分，满分5分）：

1. 毛利率壁垒：高且稳定 → 1分；波动或偏低 → 0分
2. 提价能力：能转嫁成本上涨 → 1分；不能 → 0分
3. 市场份额：稳定或提升 → 1分；下降 → 0分
4. 费用率趋势：稳定或下降 → 1分；上升 → 0分
5. 产品集中度：大单品 + 新品延展 → 1分；过度依赖或分散无焦点 → 0分

综合判定：
├── 4-5分 → 品牌护城河宽广且稳固
├── 3分   → 品牌有一定护城河但需持续投入维护
├── 1-2分 → 品牌护城河较窄，易被竞争侵蚀
└── 0分   → 无品牌护城河，本质是制造业
```
