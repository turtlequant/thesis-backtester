## 消费增长持续性

消费品企业的增长持续性是估值的核心支撑。高增长不一定好（可能靠并购或烧钱），低增长不一定差（可能进入稳定的现金牛阶段）。关键是区分增长的来源、质量和可持续性。

### 一、收入增长率计算

**步骤1：计算多维度增长率**

```
从 income 和 fina_indicator 提取近5年年度数据：

├── 1年增速 = (本期收入 - 上期收入) / 上期收入
├── 3年CAGR = (第3年收入/第0年收入)^(1/3) - 1
├── 5年CAGR = (第5年收入/第0年收入)^(1/5) - 1
│
├── 净利润增速（同口径）
├── 扣非净利润增速（排除非经常性损益干扰）
│
└── 增速趋势判定：
    ├── 加速增长：近年增速 > 3年CAGR > 5年CAGR
    ├── 减速增长：近年增速 < 3年CAGR < 5年CAGR
    ├── 稳定增长：各期增速差异 < 3pp
    └── 波动增长：增速大起大落
```

**步骤2：增长阶段判定**

| 3年CAGR | 阶段 | 含义 |
|---------|------|------|
| > 20% | 高速成长 | 估值可给更高倍数 |
| 10%-20% | 稳健增长 | 消费品理想区间 |
| 5%-10% | 成熟增长 | 关注现金分红能力 |
| 0%-5% | 低增长 | 需判断是否见顶 |
| < 0% | 负增长 | 需判断是周期还是衰退 |

### 二、有机增长 vs 并购增长

**步骤1：识别并购痕迹**

```
从 balancesheet 和 cashflow 查找并购信号：
├── 商誉变化：商誉大幅增加 → 发生重大并购
├── 投资活动现金流：
│   ├── "取得子公司支付的现金"科目大幅增加
│   └── "处置子公司收到的现金"（剥离资产）
├── 合并范围变化：收入跳升但老业务增长平稳
│
有机增长判断：
├── 无商誉增加 + 收入稳定增长 → 纯有机增长（最优）
├── 商誉增加 < 净资产5% + 收入增长 → 以有机为主
├── 商誉增加显著 + 收入跳升 → 并购驱动增长（需评估整合效果）
└── 商誉减值 → 前期并购失败信号
```

**步骤2：有机增长质量评估**

```
高质量有机增长特征：
├── 收入增长来自量增或提价（非促销补贴）
├── 新市场/新渠道渗透带来的增量
├── 新产品成功上市贡献收入
└── 不依赖大额销售费用投入

低质量增长特征：
├── 收入增长依赖销售费用大幅增加
├── 应收账款增速 > 收入增速（渠道压货）
├── 经营现金流增速远落后于收入增速
└── 毛利率持续下降（以价换量）
```

### 三、增长质量评估（经营杠杆）

**步骤1：利润增速 vs 收入增速**

```
经营杠杆 = 净利润增速 / 收入增速

判定标准：
├── 经营杠杆 > 1.5 → 强经营杠杆（规模效应显著）
│   含义：收入每增10%，利润增15%以上
│   原因：固定成本被摊薄，边际利润率提升
│
├── 经营杠杆 1.0-1.5 → 适度杠杆（健康增长）
│   含义：利润增速略快于收入
│
├── 经营杠杆 0.5-1.0 → 弱杠杆（增长但利润率被侵蚀）
│   含义：费用增速过快
│
└── 经营杠杆 < 0.5 或为负 → 无杠杆或负杠杆
    含义：收入增长但利润不增或下降
    原因：竞争加剧、成本上升、投入期
```

**步骤2：费用率趋势**

```
分析三费占收入比的趋势：
├── 销售费用率：上升 → 获客成本增加
├── 管理费用率：上升 → 管理效率下降
├── 研发费用率：上升 → 可能是正面投入（医药尤其如此）
│
优质增长：收入增长 + 费用率下降或稳定
劣质增长：收入增长 + 费用率持续上升
```

### 四、再投资率与增量回报

**步骤1：计算再投资率**

```
再投资率 = 资本支出 / 经营活动净现金流

其中：
├── 资本支出 = 购建固定资产等支付的现金（cashflow）
├── 经营活动净现金流（cashflow）
│
├── 再投资率 < 30% → 轻资产模式，大量自由现金流
├── 再投资率 30%-60% → 适度再投资
├── 再投资率 60%-80% → 重投入期
└── 再投资率 > 80% → 几乎全部现金流用于再投资
```

**步骤2：增量ROIC（新增投资回报率）**

```
增量ROIC = Δ营业利润(1-税率) / Δ投入资本

判定标准：
├── 增量ROIC > 20% → 新投资高回报（适合继续增长）
├── 增量ROIC 10%-20% → 合理回报
├── 增量ROIC < 10% → 增长在摊薄整体回报
└── 增量ROIC < WACC → 增长在毁灭价值

消费品企业理想状态：
├── 低再投资率 + 高增量ROIC → 轻资产高质量增长（最优）
├── 高再投资率 + 高增量ROIC → 重投入但回报好（成长期）
├── 低再投资率 + 低增量ROIC → 守成但空间有限
└── 高再投资率 + 低增量ROIC → 投入产出差（最差）
```

### 五、市场空间与渗透率

**步骤1：增长天花板判断**

```
间接评估指标：
├── 收入增速逐年递减 → 可能接近天花板
├── 渠道下沉（三四线城市/农村）→ 仍有渗透空间
├── 品类扩张（从主品类向相邻品类延伸）→ TAM扩大
├── 国际化（出海收入占比）→ 额外增长来源
│
人口结构影响：
├── 老龄化 → 利好医药保健、不利于年轻消费
├── 城镇化 → 利好品牌消费升级
├── 人口下降 → 总量见顶，需靠提价或升级
└── 消费降级/升级趋势 → 影响高端vs大众定位
```

### 六、增长持续性综合判定

```
持续性评分矩阵（5个维度各1分，满分5分）：

1. 增速水平：3年CAGR > 10% → 1分
2. 有机增长：以有机增长为主 → 1分
3. 增长质量：经营杠杆 > 1 → 1分
4. 再投资效率：增量ROIC > 15% → 1分
5. 增长空间：渗透率有提升空间 → 1分

综合判定：
├── 4-5分 → 增长高度可持续，估值可支撑较高倍数
├── 3分   → 增长有一定持续性，但需关注边际变化
├── 1-2分 → 增长持续性存疑，估值需谨慎
└── 0分   → 增长不可持续或已见顶
```
