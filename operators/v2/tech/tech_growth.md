## 科技成长性分析

科技企业的核心价值在于成长性。与传统行业不同，科技公司往往前期亏损、重研发投入，一旦跨过规模拐点后利润率快速提升。本算子聚焦于评估科技企业的收入增速、研发质量、经营杠杆和盈利路径。

### 一、收入增速分析

**步骤1：多维度增速计算**

```
从 income 和 fina_indicator 提取近5年年度数据：

├── 最近1年收入增速
├── 最近2个半年度收入增速（季度趋势）
├── 3年收入CAGR
├── 5年收入CAGR
│
加速/减速判断：
├── 季度收入增速逐季提升 → 加速增长（最强信号）
├── 年度增速 > 3年CAGR → 增长在加速
├── 年度增速 < 3年CAGR → 增长在减速
└── 连续2个季度增速下降 → 减速确认
```

**步骤2：科技公司增速分档**

| 收入增速 | 分档 | 估值含义 |
|---------|------|---------|
| > 50% | 超高速增长 | 可容忍亏损，关注收入质量 |
| 30%-50% | 高速增长 | 高增长溢价合理 |
| 15%-30% | 中速增长 | 需要看盈利能力改善 |
| 5%-15% | 低速增长 | 科技公司偏慢，需评估瓶颈 |
| < 5% | 停滞 | 增长逻辑需要重新审视 |

**步骤3：收入质量验证**

```
高质量收入增长特征：
├── 经营现金流增速 ≥ 收入增速（真金白银）
├── 应收账款增速 < 收入增速（非压货）
├── 预收款项/合同负债增长（下游需求强劲）
├── 大客户集中度不升反降（客户结构健康）
│
低质量增长警示：
├── 应收账款飙升（渠道塞货或赊销）
├── 现金流与利润严重背离
├── 收入依赖单一大客户（> 30%）
└── 收入靠政府补贴或非经常项目
```

### 二、研发投入强度

**步骤1：研发费用率计算**

```
研发费用率 = 研发费用 / 营业收入

从 fina_indicator 或 income 获取研发费用数据：

行业参考值：
├── 软件/SaaS：15%-30%（高投入合理）
├── 半导体：15%-25%
├── 硬件/电子：5%-15%
├── 互联网平台：10%-20%
├── AI/新兴科技：20%-40%
```

**步骤2：研发效率评估**

```
研发效率 = 收入增量 / 研发投入

评估维度：
├── 研发费用率趋势
│   ├── 上升 + 收入加速 → 研发投入正在转化为增长
│   ├── 上升 + 收入不增 → 研发可能方向错误或尚在投入期
│   ├── 下降 + 收入增长 → 研发杠杆释放（最佳阶段）
│   └── 下降 + 收入下降 → 缩减研发可能伤害未来竞争力
│
├── 资本化比例
│   ├── 研发资本化率 > 50% → 激进会计处理，利润被美化
│   ├── 研发资本化率 < 20% → 保守处理，利润更真实
│   └── 趋势上升 → 警惕粉饰利润
│
└── 研发人员占比
    ├── > 40% → 技术驱动型公司
    ├── 20%-40% → 技术+商业并重
    └── < 20% → 商业模式驱动
```

### 三、经营杠杆与边际利润率

**步骤1：经营杠杆系数计算**

```
经营杠杆系数 = 营业利润增速 / 收入增速

科技公司经营杠杆来源：
├── 软件/平台：边际成本极低，收入增加直接提升利润
├── 硬件：固定成本高（产线折旧），量增后利润率改善
├── SaaS：客户增加后获客成本被摊薄
│
经营杠杆判断：
├── 系数 > 2 → 强经营杠杆（规模效应明显）
├── 系数 1-2 → 适度杠杆
├── 系数 < 1 → 弱杠杆（成本随收入同比增长）
└── 系数为负 → 收入增长但利润恶化
```

**步骤2：边际利润率趋势**

```
边际利润率 = Δ营业利润 / Δ营业收入

分析逐年变化：
├── 边际利润率逐年提升 → 商业模式验证，接近盈利拐点
├── 边际利润率稳定且 > 0 → 已形成正循环
├── 边际利润率为负但收窄 → 亏损在改善中
└── 边际利润率为负且扩大 → 越做越亏，模式存疑
```

### 四、盈利路径分析（亏损公司专项）

**步骤1：判断是否为亏损公司**

```
如果最近1年净利润 < 0 或扣非净利润 < 0：

盈利路径评估框架：
├── 毛利率是否 > 50%？
│   ├── 是 → 产品有定价权，问题在费用端
│   └── 否 → 产品本身未建立竞争壁垒
│
├── 亏损是否在收窄？
│   ├── 亏损率逐年收窄 → 正在走向盈利
│   └── 亏损率扩大 → 烧钱加速
│
├── 何时盈利？（线性外推）
│   ├── 按当前毛利率和费用率趋势
│   ├── 估算收入达到多少时可以覆盖费用
│   └── 判断是否在2-3年内可实现盈利
│
└── 现金储备是否充足？
    ├── 现金/年度亏损 > 3年 → 跑道足够
    ├── 现金/年度亏损 1-3年 → 可能需要融资
    └── 现金/年度亏损 < 1年 → 资金链风险
```

### 五、与行业增速对比

**步骤1：相对增速评估**

```
企业增速 vs 行业增速：
├── 企业增速 > 行业2倍 → 市场份额快速提升
├── 企业增速 > 行业 → 份额温和提升
├── 企业增速 ≈ 行业 → 跟随行业成长
├── 企业增速 < 行业 → 份额流失
└── 企业增速为负 + 行业正增长 → 竞争力衰退

市场份额变化含义：
├── 份额提升 + 行业高增长 → 最优情形（量价齐升）
├── 份额提升 + 行业低增长 → 行业整合中胜出
├── 份额稳定 + 行业高增长 → 随波逐流
└── 份额下降 → 无论行业如何，都是负面信号
```

### 六、可扩展性评分

```
商业模式可扩展性评估：

高可扩展性特征（每满足一条 +1分，满3分为"高"）：
├── 毛利率 > 60%（边际成本低）
├── 经营杠杆系数 > 1.5（规模效应显著）
├── 收入增速 > 20% 且加速
├── 研发费用率下降趋势（技术壁垒已建立）
├── 预收/合同负债增长（客户锁定）

低可扩展性特征：
├── 毛利率 < 30%（本质是硬件/制造）
├── 收入增长依赖人员线性增加
├── 项目制收入（难以标准化复制）
└── 边际利润率不改善

评分：
├── 4-5条高特征 → 高
├── 2-3条高特征 → 中
└── 0-1条高特征 → 低
```
