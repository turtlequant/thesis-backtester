## 地缘政治风险排除

### 目的

地缘政治风险是前置排除条件。不满足安全要求的标的直接一票否决，无需进入后续财务分析。本算子提供系统化的地缘政治风险评估框架，包括风险分级、资产地域核查和否决机制。

### 核心原则

- 地缘政治风险属于不可对冲的系统性风险
- 资产安全优先于估值吸引力：再便宜的标的，若资产处于高风险地区，仍应排除
- 宁可错过机会，不可承受资产国有化/冻结/战争损毁等不可逆损失

---

### 五级风险分级体系

| 风险等级 | 区域/类型 | 风险描述 | 处理方式 |
|----------|-----------|----------|----------|
| **极高风险（一级）** | 非洲（除南非等少数成熟市场） | 政治不稳定，资产国有化风险，法治薄弱 | 一票否决 |
| **极高风险（一级）** | 东南亚冲突区（泰柬边境、缅甸等） | 地缘冲突频发，运营中断风险 | 一票否决 |
| **极高风险（一级）** | 敏感战略设施（特定海域港口等） | 地缘博弈焦点，国有化/制裁风险 | 一票否决 |
| **高风险（二级）** | 俄罗斯及独联体国家 | 制裁风险，资产冻结风险 | 一票否决 |
| **中等风险（三级）** | 中亚、南美部分国家、中东非核心区 | 政策多变，汇率及资本管制风险 | 需极大折价补偿（如PB<0.3），通常不建议配置 |
| **低风险（四级）** | 欧美发达市场、中国香港、新加坡、日韩 | 法治完善，产权保护成熟 | 可正常配置 |
| **极低风险（五级）** | 中国内地（尤其央国企） | 国家信用背书，政策支持 | 优先配置 |

---

### 资产地域分布核查清单

对标的执行以下核查，数据来源为年报附注及公司披露：

**第一步：收入地域拆分**
- 获取主营业务收入按地域分段数据（年报附注）
- 计算各地区收入占比
- 标记位于一级/二级风险区域的收入占比

**第二步：资产地域分布**
- 固定资产/在建工程地域分布
- 投资性房地产地域分布
- 长期股权投资（子公司/合营）注册地
- 现金及银行存款存放地（离岸vs在岸）

**第三步：供应链与关联方**
- 核心供应商集中度及所在地区
- 关联交易涉及的地域
- 关键客户地域分布

### 否决条件（任一满足即排除）

| 序号 | 否决条件 | 说明 |
|------|---------|------|
| 1 | 一级/二级风险区域资产占总资产比例 > 10% | 资产安全无法保障 |
| 2 | 一级/二级风险区域收入占总收入比例 > 10% | 收入来源不稳定 |
| 3 | 单一非发达国家市场收入占比 > 30% | 过度依赖单一高风险市场 |
| 4 | 被列入主要经济体制裁清单（美国SDN、欧盟等） | 交易/融资受限 |
| 5 | 核心业务涉及敏感技术且有出口管制风险 | 技术封锁风险 |

### 折价条件（未触发否决但需额外补偿）

| 情形 | 要求 |
|------|------|
| 三级风险区域收入占比 10-30% | 估值需在正常基础上额外折价30%以上 |
| 三级风险区域资产占比 10-20% | 估值需在正常基础上额外折价30%以上 |
| 行业面临重大监管不确定性（跨境） | 仓位上限不超过组合的5% |

### 评估决策树

```
开始：获取标的基础信息
  |
  v
步骤1：是否在主要经济体制裁清单上？
  ├── 是 -> 一票否决，结束
  └── 否 -> 继续
  |
  v
步骤2：一级/二级风险区域资产或收入占比 > 10%？
  ├── 是 -> 一票否决，结束
  └── 否 -> 继续
  |
  v
步骤3：单一非发达国家市场收入占比 > 30%？
  ├── 是 -> 一票否决，结束
  └── 否 -> 继续
  |
  v
步骤4：三级风险区域有显著敞口（10%以上）？
  ├── 是 -> 标记"需折价补偿"，记录具体敞口
  └── 否 -> 继续
  |
  v
步骤5：核心技术是否涉及出口管制领域？
  ├── 是 -> 标记"技术封锁风险"，评估替代方案
  └── 否 -> 通过地缘政治筛选
```

### 输出格式

```
标的：[名称] [代码]
地缘政治评估结果：通过 / 否决 / 需折价
风险等级：[一级至五级]
关键发现：
  - 高风险区域资产占比：X%
  - 高风险区域收入占比：X%
  - 制裁清单状态：[是/否]
  - 敏感技术涉及：[是/否]
否决原因（如适用）：[具体条件]
折价要求（如适用）：[具体要求]
```

### 注意事项

- 地缘政治格局动态变化，风险等级应结合最新国际形势调整
- 部分公司虽注册在低风险地区，但实际资产/业务集中在高风险区域，需穿透分析
- 对于资产主体在发达市场且有央企背景的标的，可视为地缘政治避险组合，给予正面评价
